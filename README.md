## 🚀 Cách sử dụng
1. Mở link app.  
2. Nhập **thông tin trường học** (Tên trường, địa chỉ, số điện thoại).  
3. Nhập **thông tin sinh viên** (ảnh 3x4, họ tên, lớp, số báo danh).  
4. Xem trước thẻ bên phải.  
5. Bấm **Tải thẻ về** để download ảnh PNG.
